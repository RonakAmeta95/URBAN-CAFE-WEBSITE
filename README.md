#  URBAN CAFE WEBSITE

A modern, responsive, and interactive café website developed using **HTML**, **CSS**, and **JavaScript**. The project provides an engaging user experience with an attractive design, online table booking, food ordering cart, and smooth animations.

---

#  FEATURES

* Fully Responsive Design (Desktop, Tablet & Mobile)
* Interactive Navigation Bar
* Mobile Hamburger Menu
* Hero Section
* Online Table Booking Popup
* About Us Section
* Café Highlights
* Promotional Banner
* Food & Beverage Menu
* Interactive Shopping Cart
* Add to Cart Functionality
* Quantity Increase & Decrease
* Remove Cart Items
* Total Price Calculation
* Contact Section
* Professional Footer with Social Media Links
* Smooth Animations using AOS Library

---

#  TECHNOLOGIES USED

* HTML5
* CSS3
* JavaScript (ES6)
* AOS (Animate On Scroll)
* Font Awesome
